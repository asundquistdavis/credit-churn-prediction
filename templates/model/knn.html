Overview

k-Nearest Neighbors (kNN) is one the simplest and popular machine learning models, which tells an algorithm to find the closest neighbors to the known data points and assign them accordingly. One way to do this is to loop through the data and calculate a score in comparison of the training dataset and the test dataset. The point where these two sets of data start to stabilize is what is used to determine the k. This also allows a user to see how accurate that model is by the corresponding accuracy score.

ROC analysis

This ROC curve is used to check the fit of the model and examine the false positive rate in comparison to the true positive rate. A curve that produces a sharp right angle is ideal with an AUC close to 1. In this example, the data is showing an ROC curve that produces the desired result we would expect but with an AUC of 85%, this also tells us that this model may not produce the best results.

Summary

The k-Nearest Neighbors model is an easy to use model that does a decent job in analyzing the data in relation to the training data, but falls a little short of the accuracy that some of the machine learning models were able to produce.
